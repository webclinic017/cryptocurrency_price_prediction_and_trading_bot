* Folder information:
    + normalized_features: data with features normalized from original features 
    + original_features: original features generated by feature engineering phase
    + raw: raw data crawled from web
        + future: future type data
        + spot: spot type data
    + others: others data
    
* File name convention:
    + normalized_features: norm_[future | spot]_[source(s)]_[token | stock | symbol | token pairs].csv  
    + original_features: original_[future | spot]_[source(s)]_[token | stock | symbol | token pairs].csv
    + raw: raw_[future | spot]_[source]_[token | stock | symbol | token pairs].csv
    + others: others data
